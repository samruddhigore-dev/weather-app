# 🌦️ Weather App

A simple and responsive Weather App built using HTML, CSS, and JavaScript.  
It fetches real-time weather data using an API and displays temperature, humidity, and wind speed for any city.

## 🚀 Live Demo
🔗 https://samruddhigore-dev.github.io/weather-app/

## ✨ Features
- 🌍 Search weather by city name
- 🌡️ Real-time temperature display
- 💧 Humidity information
- 🌬️ Wind speed details
- 🎨 Clean and modern UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Weather API
